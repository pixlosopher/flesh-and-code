# Flesh & Code: Arrows and Axioms

> *"Touch a screen and the screen fingerprints you back."*

A digital phenomenology exploring the ontological intersection of bodies, images, and code as one reversible tissue.

---

## About

**Flesh & Code** is a philosophical text co-authored with AI, written in Nietzsche's aphoristic tradition. The book contains **320 arrows** (aphorisms) across **7 thematic axes**, exploring the chiasmic relationship between human flesh and digital code.

This repository contains the website, assets, and documentation for the project.

**Live Site**: [fleshandcode.art](https://fleshandcode.art) (via GitHub Pages)

---

### Seven Thematic Axes

| # | Theme | Concept |
|---|-------|---------|
| I | **Encroachment** | Porous boundaries between flesh and technology |
| II | **Passivity** | Receptivity and latency as creative forces |
| III | **Chiasmic Entanglement** | Reciprocal seeing/being-seen |
| IV | **Reversible Time** | Digital temporality as malleable clay |
| V | **Visibility/Invisibility** | Surfacing hidden infrastructures |
| VI | **Worlding** | Networks of meaning and interpretation |
| VII | **Creation & Destiny** | Co-authorship with algorithms and glitch |
