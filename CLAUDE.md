# Flesh and Code

A digital phenomenology project exploring the ontological intersection of bodies, images, and code as one reversible tissue.

## Project Overview

This is a creative web experience combining philosophy, glitch aesthetics, and interactive digital art. The project manifests the ideas from "Flesh and Code: Arrows and Axioms" - a philosophical text drawing on Merleau-Ponty's phenomenology, Nietzsche's aphoristic style, and contemporary media theory (Simondon, Stiegler, Sloterdijk).

**Live site:** GitHub Pages
**Repository:** https://github.com/pixlosopher/flesh-and-code

## Core Concepts

The project explores seven thematic axes:

1. **Encroachment** - Porous boundaries between flesh and technology ("Hybrid Skin")
2. **Passivity** - Receptivity and latency as creative forces ("Composed Latency")
3. **Chiasmic Entanglement** - Reciprocal seeing/being-seen ("Feedback Compose")
4. **Reversible Time** - Digital temporality as malleable clay ("Temporal Plasticity")
5. **Visibility/Invisibility** - Surfacing hidden infrastructures ("Critical Surfacing")
6. **Worlding** - Networks of meaning and interpretation ("Hermeneutic Worlds")
7. **Creation & Destiny** - Co-authorship with algorithms and glitch ("Co-Authoring Destiny")

## Design Philosophy

When working on this project, embrace these principles from the text:

- **Surface the pipes** - Make at least one layer of infrastructure visible
- **Design for passivity** - Include moments of stillness and waiting
- **Signal entropy** - Let compression artifacts and glitch reveal the medium
- **Latency as lyric** - Treat delays and loading as compositional elements
- **Reciprocal gaze** - The interface should look back at the viewer

## Aesthetic Guidelines

- **Glitch aesthetics** - Embrace JPEG artifacts, datamosh, visual corruption as features
- **Dark themes** - The philosophical text uses dark backgrounds with light text
- **Temporal loops** - GIF-like repetition, video loops as meditation
- **Hybrid materiality** - Blur the line between physical and digital textures
- **Typography** - Text as visual element, aphoristic fragments

## Development Notes

- The project is intentionally experimental and artistic rather than conventionally functional
- Errors and glitches may be intentional aesthetic choices
- Performance quirks (lag, loading) can be features, not bugs
- The experience should feel embodied and responsive to user presence

## Working with This Project

When making changes:
1. Preserve the philosophical/artistic intention behind existing code
2. Consider how changes affect the phenomenological experience
3. Document any intentional "errors" or glitch effects
4. Test on multiple devices - embodied experience varies by hardware
5. Keep media assets optimized but allow for intentional compression artifacts
